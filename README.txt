INTRODUCTION

Oxyus is an open source search engine written 100% in java,
aimed to provide a search button for your website in an easy
way.

The Oxyus Web Site is at:
	http://oxyus.sourceforge.net/

INSTALLATION

	
1. Deploy the oxyus.war file in a java complaint
 server like Apache Tomcat or JBoss.
	
	
2. Go to http://yourserver/oxyus/admin/ to instruct Oxyus which website to index.


  For specific server instructions review the DEPLOY.txt file

BUILDING

For building Oxyus you need to use Apache Ant with the build.xml
file.
